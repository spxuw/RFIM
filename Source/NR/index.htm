Numerical Recipes in C++ and C
Versions 2.11 (C++) and 2.10 (C)

This page gives top-level access to all of the Numerical Recipes C++ and C source code files. There are three language versions included in this release, accessed by the links below:

  • Numerical Recipes in C++
  • Numerical Recipes in C (ANSI C)
  • Numerical Recipes in C (K&R C)

You can access the files in this release either through your web browser and these linked pages, or else by looking in the actual file directories/folders below this one. The files are all plain text files with conventional file names/extensions.
